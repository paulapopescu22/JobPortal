# Task App 📝

A simple full-stack task management application built with **Node.js + Express** (backend) and **HTML/CSS/JS** (frontend).

---

## 🔹 Features

- View existing tasks  
- Add new tasks  
- Simple REST API (`GET /api/tasks`, `POST /api/tasks`)  
- Responsive and user-friendly interface  

---

## 🔹 Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript ES6+  
- **Backend:** Node.js, Express.js  
- **CORS** for cross-origin requests  
- **JSON** for data exchange
